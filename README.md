## ✨ Возможности

- **Точные данные игры**  
  Встроены официальные таблицы стоимости и времени улучшения для **всех шести героев** (уровни 1–105), взятые из проверенных источников.

- **Выбор ратуши (TH 7–18)**  
  Автоматически отображаются только герои, доступные на вашем уровне ратуши, с корректными максимальными уровнями.

- **Золотой пропуск (20% скидка)**  
  Скидка применяется и ко времени, и к стоимости улучшений. Процент можно изменить в настройках.

- **Параллельная прокачка**  
  Можно указать, сколько героев улучшаются одновременно. Внутри одного героя уровни идут строго последовательно, разные герои стартуют параллельно (если хватает «строителей»).

- **Защита от ночных финишей**  
  Настройте запрещённый интервал (по умолчанию 00:00–07:00 МСК). Планировщик сдвинет старт так, чтобы улучшение не заканчивалось в это время.

- **Группировка по героям**  
  Таблицы «Запланированные улучшения» и «График прокачки» сгруппированы по персонажам, блоки можно сворачивать для удобства.

- **Реальное время до завершения**  
  Для каждого этапа отображается таймер с обратным отсчётом (дни, часы, минуты, секунды).

- **Ручное редактирование старта**  
  Время начала любого улучшения можно изменить прямо в таблице — расписание автоматически пересчитается.

- **Сохранение прогресса**  
  Все настройки и текущий план автоматически сохраняются в `localStorage` браузера. При повторном открытии страницы данные восстанавливаются.

- **Адаптивный дизайн**  
  Удобно использовать как на компьютере, так и на мобильных устройствах.

## 🧭 Как использовать

1. **Выберите ратушу** – отобразятся доступные герои и их максимальные уровни.
2. **Укажите цели** – для каждого героя задайте текущий уровень и желаемый целевой. Нажмите «Добавить в план».
3. **Настройте параметры**:
   - Запретный интервал (например, с 00:00 до 07:00 МСК).
   - Количество одновременно улучшаемых героев.
   - Процент скидки (по умолчанию 20%).
   - Начальную дату и время планирования.
4. **Нажмите «Рассчитать расписание»** – появится подробный график с датами старта и финиша, стоимостью, длительностью и статусом.
5. **Редактируйте при необходимости** – кликните по времени старта в таблице, чтобы изменить его вручную. Последующие строки пересчитаются автоматически.
6. **Сворачивайте группы** – клик по заголовку героя скрывает его уровни, чтобы не перегружать экран.

> 💡 *Все изменения плана сразу сохраняются в вашем браузере. Чтобы сбросить прогресс, нажмите «Сбросить цели».*

## 💻 Установка и запуск

Проект состоит из одного HTML‑файла, не требует сервера или сборки.

### Локально

1. Скачайте репозиторий или просто сохраните файл `index.html`.
2. Откройте его в любом современном браузере (Chrome, Firefox, Safari, Edge).

### Размещение на GitHub Pages

1. Создайте новый репозиторий на GitHub.
2. Загрузите файл `index.html` в корень репозитория.
3. Перейдите в **Settings → Pages**, выберите ветку `main` и папку `/ (root)`.
4. Через минуту сайт станет доступен по адресу `https://ваш_юзернейм.github.io/имя_репозитория/`.

## 📊 Структура данных

Все цифры (стоимость и время улучшения) вшиты в JavaScript‑объекты на основе официальных таблиц игры.  
Использованы следующие источники:

- Основная таблица уровней героев (1–105) из проверенных гайдов.
- Максимальные уровни героев для каждой ратуши (TH7–TH18).

Данные актуальны на **апрель 2026 года**. При выходе новых обновлений их можно легко дополнить в коде (массивы `barbarianRaw`, `archerRaw` и т.д.).
